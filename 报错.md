### 注意事项
- 已注册的旧用户需要手动创建学生档案或重新注册
- 新用户注册时必须提供 class_id，否则学生档案创建会失败
- 可以为现有用户批量创建学生档案：
  ```sql
  INSERT INTO student (user_id, student_number, class_id, enrollment_date, status, deleted)
  SELECT u.id, CONCAT('STU', u.id), COALESCE(u.class_id, 1), CURDATE(), 1, 0
  FROM user u
  LEFT JOIN student s ON u.id = s.user_id
  WHERE s.id IS NULL AND u.id IN (
    SELECT ur.user_id FROM user_role ur
    JOIN role r ON ur.role_id = r.id
    WHERE r.role_code = 'USER'
  );
  ```
